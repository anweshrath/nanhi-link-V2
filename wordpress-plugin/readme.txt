== Installation ==

### Automatic Installation

1. Go to your WordPress admin dashboard
2. Navigate to Plugins ‚Üí Add New
3. Search for "Nanhi.Links Integration"
4. Click "Install Now" and then "Activate"

### Manual Installation

1. Download the plugin zip file
2. Upload it to your `/wp-content/plugins/` directory
3. Extract the files
4. Activate the plugin through the WordPress admin

### Setup

1. Go to Nanhi.Links ‚Üí Settings in your WordPress admin
2. Sign up for a free account at [nanhi.link](https://nanhi.link)
3. Generate your API key from your account settings
4. Enter the API key in the plugin settings
5. Configure your preferences and start creating links!

== Frequently Asked Questions ==

= Do I need a Nanhi.Links account? =

Yes, you need a free Nanhi.Links account to use this plugin. You can sign up at [nanhi.link](https://nanhi.link).

= Is there a free plan? =

Yes! Nanhi.Links offers a generous free plan that includes:
* 1,000 links per month
* Basic analytics
* Custom short codes
* API access

= Can I use my own domain? =

Yes, premium plans include custom domain support for branded short links.

= Will my links work if I deactivate the plugin? =

Yes, all links created through Nanhi.Links will continue to work even if you deactivate the WordPress plugin.

= Can I import existing short links? =

Yes, the plugin automatically syncs with your existing Nanhi.Links account and displays all your links.

= Is there bulk link creation? =

Yes, you can create multiple links at once using the bulk creation feature.

= Can I track clicks in real-time? =

Yes, the plugin provides real-time click tracking and analytics.

= Is the plugin translation ready? =

Yes, the plugin is fully translation ready and includes translation files.

This plugin connects to the Nanhi.Links service to create and manage short links. Please review the [Nanhi.Links Privacy Policy](https://nanhi.link/privacy) for information about data handling.

The plugin stores minimal data locally:

No personal data is stored locally beyond what's necessary for functionality.
